# OfferPilot

本地优先的 AI 求职作战台。

OfferPilot v0.1 是一个面向 Boss 直聘求职场景的本地 AI 求职手账。它不接 API，不做自动投递，不做爬虫，不做后端。它帮助求职者把“看岗位、生成分析 Prompt、承接外部 AI 结果、提取 Boss 打招呼话术、保存岗位台账、跟踪沟通状态”整理成一条稳定流程。

## v0.1 定位

OfferPilot v0.1 不是 AI 自动分析器，而是：

> 本地优先的 AI 求职工作流容器。

它负责：

1. 标准化简历 / 项目 / 求职偏好输入
2. 标准化岗位 JD 输入
3. 生成结构化 AI 分析 Prompt
4. 承接 ChatGPT / Claude / Gemini 返回的分析结果
5. 保存完整 AI 原文
6. 尽力提取 Boss 打招呼话术
7. 沉淀岗位分析报告
8. 管理岗位沟通状态

真正的模型推理由用户自带外部 AI 完成。

## v0.1 不做

1. 不接 OpenAI / Claude / Gemini API
2. 不做 BYOK
3. 不做后端
4. 不做登录注册
5. 不做 Boss 自动投递
6. 不做 Boss 爬虫
7. 不做浏览器插件
8. 不做 PDF / Word 简历解析
9. 不做云端同步
10. 不做复杂数据统计
11. 不做完整 AI Chat

## 版本路线

- v0.1：Manual Mode，不接 API，验证求职工作流
- v0.1.5：Personal BYOK，仅个人本地验证自动分析
- v0.2：正式 AI Adapter，延后

## 开发纪律

> 配置存一份，岗位分条存，原文先落盘；先建能存能列，再做进料回流，最后收状态与认知。P0 之外，一个都不碰。
